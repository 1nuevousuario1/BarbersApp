# BarbersApp
Aplicación web para barbería que permite a los clientes seleccionar cortes de cabello, agendar citas por bloques de tiempo disponibles, y enviar pedidos vía WhatsApp.Cumple con estándares de usabilidad,seguridad,escalabilidad y compatibilidad móvil.✅¿Por qué funciona bien?Explica qué hace la app (catálogo+citas +WhatsApp).Es profesional y original.
